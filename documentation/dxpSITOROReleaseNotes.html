dxpSITORO Release Notes

Release 1-1 (28-June-2017)

This is built with Handel 1.1.19 which is required for the new FalconX firmware.

Added suppport for SCATriggerMode and SCAPulseDuration.

Added suppport for RisetimeOptimization.

Fixed IgnoreGate and InputLogicPolarity so they are set for normal MCA mode, not just mapping modes.

Release 1-0 (7-June-2017)

Initial release. Supports mca spectrum and mca mapping modes on the FalconX modules. This is built with Handel 1.1.14
